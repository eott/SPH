# Installation
1. ```git clone path/to/repo```
2. ```cd SPH```
3. ```mkdir include```
3. ```mkdir build```
4. ```cd include```
5. ```git clone path/to/yaml-cpp-repo```
6. ```cd ../build```
7. ```mkdir output```
7. ```mkdir output/ascii```
7. ```mkdir output/vtk```
7. ```cmake ./../```
8. ```make -j```
9. ```cp ../sample.param sample.param```
